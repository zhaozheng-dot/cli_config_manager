# cli_config_manager
一个命令行工具（CLI），用于读取JSON或YAML等结构化配置文件，将其转化为 Python 可处理的对象来进行数据清洗。

- 这是 AI 的基础

    OpenAI 的 Function Calling 和 LangChain 的 Tool 定义全部基于 Pydantic。不懂 Pydantic，就看不懂如何定义一个 Agent 工具。
